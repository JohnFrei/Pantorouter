Pantorouter plans 1:1 scale parts drawings

This page contains 1:1 printable drawings of the parts. You may also optionally print 1:1 views of the assembled machine

To print the drawings to come out as full size 1:1 scale drawings with your browser, you will need to set your top, left and right margins to zero, and the bottom margin to less than 0.75" or 2 cm. When you print, do NOT scale the output (browser will often scale by default). The print scale should be set to 100%.

It's best to try printing just the first page to see how it works out. The border surrounding the images should be 19 x 25 cm in size. If it's any smaller than that, the drawing is not scaled correctly.

You can click on any of the images below for an enlarged view, but to print out the images, please see the instructions below.

Using the BigPrint program

If you are using a Windows PC, you can also download the free evaluation copy of BigPrint, then drag any of the images below onto the program. The evaluation version will recognize these images and act like the full version while one of the images is loaded. You don't need to buy the BigPrint program to print these plans.

Tweaking the scale

Most InkJet printers will print very accurately without making any adjustments. However, some laser printers will print off-scale by as much as 1%. If you need to adjust the scale, print one of the images and measure the actual size of the border and calculate the scale error. Measure the borders to the outside edges of the lines.

For example, if the border is 24.5 cm wide, that would be 24.5/25 = 98%. To compensate, set the print scale (in your browser, or PDF viewer) to 102%.

Also check the border in the other direction, making sure it's 19 cm. If you are using my BigPrint program, you can make very fine adjustments to the scale, both horizontally and vertically.
